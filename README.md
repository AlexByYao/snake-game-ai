# 贪食蛇游戏

一个使用 HTML5 Canvas 和 JavaScript 开发的经典贪食蛇游戏。

## 功能特点

- 流畅的游戏体验
- 实时分数统计和最高分记录
- 支持暂停/继续功能
- 精美的视觉效果和渐变背景
- 响应式键盘控制
- 本地存储最高分记录

## 如何游玩

1. 打开 `index.html` 文件在浏览器中运行游戏
2. 使用方向键 ↑ ↓ ← → 控制蛇的移动
3. 按空格键可以暂停/继续游戏
4. 吃到红色食物获得分数，每吃一个食物得10分
5. 避免撞墙和撞到自己的尾巴

## 游戏截图

游戏界面采用现代化的渐变设计，具有发光效果和圆角设计。

## 技术栈

- HTML5 Canvas
- CSS3 (渐变、阴影、毛玻璃效果)
- 原生 JavaScript (ES6+)

## 本地运行

直接在浏览器中打开 `index.html` 即可开始游戏。
